## Session 1: ES6+ Deep Dive

### Topics Covered
- Let/Const, Arrow Functions, Template Literals
- Destructuring, Spread/Rest Operators
- Modules (Import/Export)

### Tasks
- **Lab**: Refactoring legacy JS code to ES6+
- **Task**: Build a modular JS utility library

---

## Session 2: Async Programming & DOM

### Topics Covered
- Asynchronous JS: Promises, Async/Await
- Fetch API & JSON Handling
- DOM Manipulation & Event Loop

### Tasks
- **Lab**: Fetching data from a public API (JSONPlaceholder)
- **Task**: Create a dynamic list renderer using raw JS

---

## Session 3: React.js Architecture

### Topics Covered
- Introduction to SPA & Virtual DOM
- create-react-app vs Vite
- JSX Syntax & Component Structure

### Tasks
- **Lab**: Environment Setup (Node, NPM, Vite)
- **Task**: Build a static UI skeleton using JSX
